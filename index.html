Welcome!

Wallabagger is extension for add pages to wallabag, with ability to:

  • save current page
  • edit title
  • add (with autocomplete!) and remove tags
  • set starred and archived
  • delete

What is wallabag?

wallabag is a self hostable application allowing you to not miss any content anymore. More informations on website: wallabag.org and github.com

Requirement

  • Wallabag v.2 on server side.
  • Chrome-based browser
